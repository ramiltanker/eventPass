# 🚀 EventPass

Backend-приложение на **Node.js + NestJS**.

---

## 📦 Требования

- Node.js >= 18
- npm >= 9

Проверить версии:

```bash
node -v
npm -v
```

## ⚙️ Требования

```bash
git clone <repo-url>
cd my-backend
npm install
```

## 🧪 Требования

```bash
npm run start:dev
```

Сервер будет доступен по адресу:

```bash
http://localhost:3000
```

Проверка health endpoint:

```bash
GET http://localhost:3000/health
```

## 🏗 Production сборка

Сборка проекта:

```bash
npm run build
```

Запуск production версии:

```bash
npm run start:prod
```
